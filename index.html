The Hayat

The Hayat

Background

The Hayat is a social impact business I previously had. The Hayat aims to work with small communities of artisans around the world to bring their products to a wider audience.

The Hayat launched in 2018 with a website created through Squarespace. At the time I wanted to incorporate an interactive world map showing the countries our artisanal products come from but was sure how to produce this. Therefore, it felt right to create this as a front-end project.

Currently, this is an information website giving information around the products, cultures and traditions of the artisans that create the Hayat products, however this is a work in progress project with the aim of being built into a full ecommerce site.

To create the interactive world map, I used an SVG file which I turned into xml code which I added to the html file. I then used the Add Event Listener tool in JavaScript to create the functionality that allows country names to to appear when the mouse hovers over each country.

Tools / Frameworks

HTML, CSS, JavaScript
